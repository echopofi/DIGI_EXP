# DIGI_EXP

DIGI_EXP is a modern webpage currently under development. The goal is to provide DIGI Expressions with a clean, professional online presence while showcasing modern frontend architecture and scalable application design.

## 🚧 Project Status

> **Work in Progress**

The frontend is actively being developed. Backend services and authentication will be added in future iterations.

## ✨ Planned Features

- Modern landing page
- Responsive design
- Service showcase
- Contact section
- Authentication
- Admin dashboard
- API integration

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## 🌐 Live Demo

**Live Demo** https://digi-exp-2xvk.vercel.app/

## 🚀 Running Locally

```bash
git clone https://github.com/echopofi/DIGI_EXP
cd DIGI_EXP
npm install
npm run dev
```
